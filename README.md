# 🎮 Арканоид: Царство Пяти Начал

Классический арканоид с элементами narrative-driven progression, реализованный на стеке:
- **Phaser 4** (игровой движок)
- **Vite 8** (сборщик)
- **TypeScript** (строгая типизация)

## 📦 Установка
```bash
npm install
```

## 🚀 Запуск
```bash
npm run dev
```

## 📖 Документация
Пошаговая реализация проекта доступна в логах коммитов и комментариях кода.
